AllDataR Documentation

Download DWD station data from CDC-ftp server.

Description

AllData retrieves data from the CDC ftp-server using Raffaels function GetData and the dummy metadata as obtained from the above function. Using these two functions instead of his complete package is necessary because it was eveloped to run under Windows, and it was developed to have a user interface. Here, I have hardcoded to download all available daily climate data. I allow to choose the station via its station id. I want the output data frame to also hold information about the station name, longitude, and latitude which are input to this funciton.

Usage

AllData(station.id, station.name, station.lat, station.lon, daily = TRUE,
  download = FALSE, verbose.DWD = TRUE)

Arguments

station.id

a string of length 5 containing the station id as provided in the metadata of the downloaded station

station.name

string of the station name corresponding to the above station id

station.lat

latitude of the station

station.lon

longitude of the station

daily

optional boolean to specify whether station data has daily or hourly time steps. Default is daily (TRUE).

download

optional boolean to specify whether only downloading shall be performed (TRUE) or unzipping and data extraction shall be performed as well (FALSE). Default is perform unzipping and extraction also (FALSE).

verbose.DWD

optional boolean to specivy whether to print out what's going (TRUE) on or surpress it (FALSE). Default is to print out messages (TRUE).

Value

data1 a data frame (returned as list) containing all this data including windspeed listed chronologically
